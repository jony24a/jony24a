# Proyecto Base
> Author: Jonatan Andres Novoa Rodriguez 

## Enfoque 
En su labor como programador de software ha sido el elegido para el desarrollo de un programa que calcule las ventas de monitores, pidiendo como primera base el nombre de la persona quien compra el producto, luego pidiendo sus datos personales, como es el número de cédula, número celular, despues de eso que aparezca la marca de los monitores, como lo que es LG, SAMSUNG, ASUS, APPLE, etc., cuantos va a llevar y que pueda ver los valores de cada producto,  y luego realizar la suma total de todos los productos que va a llevar, y a lo ultimo aparezcan los datos de la persona los productos que va a llevar y su cuenta total. 

+ El programa tiene que hacer la suma de los productos que el cliente vaya a llevar.
+ Cada uno de los productos que se ofrezcan debe tener su precio para la información.
